How to Play

  1. You will see several empty boxes—each box needs a number from 1 to 9.
  2. Enter a single digit number (1 through 9) in each empty box.
  3. Between the boxes, you’ll see mathematical operators like plus (+) and multiply (*). These operators form equations using the numbers you enter.
  4. Your goal is to make every equation created by these operators equal the target number displayed at the top of the game.
  5. For example, if the target number is 12, then the sum, product, or other operation of the numbers on each row, column, and diagonal should be exactly 12.
  6. If any equation does not equal the target number, you will need to adjust the numbers until all equations are correct.
  7. Once you think you’ve got all the equations equal to the target number, click the Check button to see if your solution is correct.
  8. If the message says "Correct!", you’ve solved the puzzle! If not, try different numbers and check again.

Take your time and use logic and trial-and-error to find the right combination. Good luck!
